# 故事論壇系統 - 讀者指南

故事論壇提供訂閱功能，讓你不錯過喜愛故事的任何更新！

## 如何使用

### 方式 1：透過訂閱入口（推薦）

1. 在故事討論串中找到訂閱入口訊息（通常在開頭）
2. 點擊對應的按鈕選擇訂閱類型

**三種訂閱類型：**
- **📘 Release（正式發布）** - 接收正式章節更新通知
- **🧪 Test（測試/預覽）** - 接收草稿、測試版本通知
- **⭐ 關注作者** - 自動訂閱該作者在此伺服器的所有故事

### 方式 2：使用指令

在討論串中輸入：
```
/sf subscribe type:release
```

**參數說明：**
- `type` 可選：`release`、`test`、`author_all`
- 不填寫 `type` 則預設為 `release`

### 查看所有訂閱

想知道自己訂閱了哪些故事？
```
/sf view
```
系統會列出你在此伺服器所有訂閱的故事，並標示訂閱類型。

### 取消訂閱

如果不想再追蹤某個故事：
```
/sf unsubscribe
```
在該討論串中使用此指令，會移除該討論串的所有訂閱類型。

## 訂閱後會發生什麼？

### Release 通知
當作者發布正式章節並使用 `/sf notify type:release` 時：
- 你會在討論串中被 @ 提醒
- 通知訊息會包含更新內容的連結
- 可以直接點擊跳轉到最新更新

### Test 通知
當作者發布測試內容並使用 `/sf notify type:test` 時：
- 只有訂閱 Test 的讀者會收到通知
- 適合願意提供反饋的讀者
- 可以參與作者的創作過程

### 關注作者
選擇「關注作者」後：
- 該作者在此伺服器創建新故事討論串時
- 如果作者開啟了訂閱功能，你會自動訂閱
- 方便追蹤喜愛作者的所有作品

## 訂閱類型建議

### 只想追正式更新
✅ 訂閱 **Release**
- 只接收正式章節通知
- 不會被測試版本打擾

### 願意提供反饋
✅ 同時訂閱 **Release** 和 **Test**
- 可以看到草稿和測試版
- 幫助作者改進內容
- 提前閱讀新內容

### 追蹤特定作者
✅ 選擇 **關注作者**
- 自動訂閱該作者的所有新作品
- 不錯過任何創作

**小提示：** 你可以在單一討論串中同時訂閱多種類型！

## 常見問題

**Q: 可以同時訂閱多個故事嗎？**
A: 可以！你可以訂閱任意數量的故事，使用 `/sf view` 查看列表。

**Q: 訂閱會自動取消嗎？**
A: 不會，訂閱會一直保留直到你主動使用 `/sf unsubscribe` 取消。

**Q: 所有討論串都能訂閱嗎？**
A: 只有在故事論壇頻道中的討論串才能使用訂閱功能。

**Q: 如果作者刪除了訂閱入口訊息怎麼辦？**
A: 仍然可以使用 `/sf subscribe` 指令訂閱，不影響功能。

**Q: 可以只訂閱 Test 不訂閱 Release 嗎？**
A: 可以！使用 `/sf subscribe type:test` 即可。

**Q: 關注作者會自動訂閱舊故事嗎？**
A: 不會，只會自動訂閱關注之後創建的新討論串。

**Q: 如何取消關注作者？**
A: 在每個討論串中使用 `/sf unsubscribe`，或是不再訂閱該作者的新作品。

**Q: 作者會知道有誰訂閱嗎？**
A: 作者只能看到訂閱人數統計，無法查看具體名單。

## 指令速查表

| 指令 | 說明 | 範例 |
|------|------|------|
| `/sf subscribe` | 訂閱故事更新 | `/sf subscribe type:release` |
| `/sf unsubscribe` | 取消訂閱 | `/sf unsubscribe` |
| `/sf view` | 查看所有訂閱 | `/sf view` |

## 使用技巧

### 管理訂閱列表

定期使用 `/sf view` 檢視訂閱列表：
- 確認正在追蹤哪些故事
- 及時取消不再感興趣的訂閱
- 保持列表整潔

### 選擇合適的訂閱類型

根據你的閱讀習慣選擇：
- **輕度閱讀** → 只訂閱 Release
- **深度參與** → 同時訂閱 Release 和 Test
- **作者粉絲** → 選擇關注作者

### 訂閱入口的便利性

如果討論串有訂閱入口：
- 可以一鍵切換訂閱類型
- 查看各類型的訂閱人數統計
- 更方便的訂閱體驗

## 禮儀建議

1. **尊重作者的創作**
   - 收到 Test 通知時，提供建設性的反饋
   - 不要劇透測試版內容給其他讀者

2. **管理通知**
   - 如果不想再追蹤某故事，請及時取消訂閱
   - 避免累積大量不必要的訂閱

3. **善用關注作者功能**
   - 確實喜歡該作者的作品再選擇關注
   - 可以隨時在個別討論串取消訂閱

開始訂閱你喜歡的故事，不再錯過精彩更新！
