# Server Info Command

Display detailed information about the current server in a visually appealing card format.

## Usage

```
/serverinfo
```

## Description

Generates a custom image card containing:
- Server Banner & Icon
- Server Name & ID
- Owner information
- Verification status
- Description & Tags
- Statistics (Members, Boosts, Channels, Roles, Emojis)
- Specifications (Locale, Verification Level, Created Date)

## Examples

`/serverinfo` - Displays the card for the current server.
